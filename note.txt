# Gameplay

Le jeu contient deux modes :
- Le mode map
- Le mode interaction

## Mode map

Le joueur se déplace sur une map. Sur la map il ne peut rien faire d'autre que se déplacer dans 4 directions.

La map contient des cases 'wall' sur lesquelles le joueur ne peut pas aller

Sur la map sont placés des objets. Si le joueur se déplace sur un objet le jeu passe en mode interaction

## Mode interaction

Le joueur lis un texte, il peut choisir des actions dans un menu ces actions peuvent avoir une influence sur l'état d'objets.

enfin ces actions peuvent faire basculer le jeu en mode map (quitter l'interaction)
